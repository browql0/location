# Architecture

Phase 1 pose la fondation technique du SaaS:

- monorepo `frontend` et `backend`;
- API REST Express sous `/api/v1`;
- Prisma avec PostgreSQL;
- schéma multi-tenant complet;
- seed minimal Super Admin et plans SaaS;
- frontend React/Vite avec routing, thème et layout vide.

Les modules métier seront développés dans les phases suivantes.
